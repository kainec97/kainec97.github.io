Object Oriented Program Development (C#/Visual Studio)
Assignment 1 - Menu

This assignment was the first for this OOP unit. It was undertaken in Semester 1 of my second year studying my Bachelor of IT, and involved the creation of a simple command prompt menu by the user. The user is prompted to enter in the menu header and items, which is then formatted and displayed. Finally, the user is prompted to select an item from the menu, which is then displayed back to them in a sentence. The user is given the option to restart the program once this is complete.

Interactive Multimedia Application Development II "Space Invaders" Game

This assignment had us conceptualise and write up design documentation for a "Space Invaders"-like game, which we would use Google Web Designer and JavaScript to create. We were given some tutorials on how to program various game mechanics (collision checking, player/enemy movement, player/enemy fire). We were also required to add a feature to the game that we designed and programmed ourselves. In this case, I added a shield to the game, which is destroyed after being hit a few times by enemies.

Managing Software Development Assignment 1 - Acceptance Tests, Black-box Tests, Configuration Management & RFP

This report required us to write about acceptance tests (including writing up an acceptance test plan, with risk mitigation and acceptance test derivation). The report also included a detailed black-box testing plan for the Microsoft Word 2016 print dialog/screen. Part 2 of the assignment required us to write a short essay on configuration management. This included code/version management and build management strategies. Finally, Part 3 of the report had us write a short Request for Proposal (RFP) for a fictional company.
